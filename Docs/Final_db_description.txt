1️⃣ ACCOUNTS
Mục đích: Lưu thông tin xác thực và phân quyền người dùng.
Field
	Mô tả
	email
	Email đăng nhập, duy nhất
	password
	Mật khẩu đã hash (null nếu OAuth)
	role
	Quyền hệ thống: user, admin
	________________

	Mô tả

	session_id
	Định danh phiên kết nối
	connected

	________________

9️⃣ MATCH_QUESTION
Mục đích: Snapshot câu hỏi theo từng round (quan trọng để replay & audit).
Field
	Mô tả
	match_id
	Trận đấu
	round_no
	Số round (1–4)
	round_type
	MCQ, FILL, WHEEL, BONUS
	question_idx
	Thứ tự câu trong round
	question
	Dữ liệu câu hỏi (JSON)
	📌 Unique: (match_id, round_no, question_idx)
________________

1️⃣1️⃣ QUESTIONS
Mục đích: Ngân hàng câu hỏi dùng để sinh match.
Field
	Mô tả
	type
	MCQ, BID, WHEEL, BONUS
	data
	Nội dung câu hỏi
	active
	Có được phép dùng không
	Ví dụ
MCQ:
{
  "type": "MCQ",
  "question": "Giá của iPhone 15 Pro tại Việt Nam là bao nhiêu?",
  "choices": [
        "25.990.000",
        "27.990.000",
        "29.990.000",
        "31.990.000"
  ]
}
	BID
{
  "question": "Giá gốc của PS5 Digital Edition (USD)?",
  "correct_answer": 399
}
	WHEEL
{
  "segments": [
        { "value": 0,   "weight": 40 },
        { "value": 50,  "weight": 30 },
        { "value": 100, "weight": 20 },
        { "value": 200, "weight": 10 }
  ]
}
	BONUS
{
  "question": "Giá của iPhone 15 Pro tại Việt Nam?",
  "real_price": 29990000,
  "display_rule": {
        "min_offset": -5000000,
        "max_offset": 5000000
  }
}
